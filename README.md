# S.H.I.E.L.D. — Avengers Personnel Database

Este proyecto es una interfaz táctica inspirada en el universo Marvel, desarrollada para la asignatura de Diseño de Interfaces Web (DIW).

## 🚀 Versión Publicada
Puedes ver el proyecto en funcionamiento aquí:
**[https://ececs.github.io/DIW-Proyecto-final/](https://ececs.github.io/DIW-Proyecto-final/)**

## 📂 Características
- **Interfaz Táctica**: Estética HUD (Head-Up Display) con efectos CRT y escaneado.
- **Terminal Interactivo**: Animación de acceso a archivos secretos de S.H.I.E.L.D.
- **Base de Datos de Agentes**: 8 perfiles detallados de los Vengadores con analíticas de poder.
- **Multimedia**: Integración de audio y video del sistema.
- **Responsive Design**: Optimizado para escritorio, tablets y dispositivos móviles.

## 🛠️ Tecnologías
- HTML5 Semántico
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- JavaScript Vanilla (Interacción y manipulación del DOM)
